# Foro Universitario en PHP

Proyecto personal para crear un foro universitario donde los estudiantes pueden:
- Registrarse e iniciar sesión
- Crear publicaciones académicas y no académicas
- Comentar publicaciones
- Subir archivos

## Tecnologías
- PHP 8
- Nginx
- MariaDB
- HTML/CSS
- Arquitectura MVC simple

## Instalación
1. Clonar repositorio
2. Configurar base de datos
3. Copiar `database.example.php` a `database.php`
4. Configurar Nginx
5. Ejecutar en localhost
